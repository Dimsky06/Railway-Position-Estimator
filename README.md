## üßê About The Project
In public transportation systems, knowing the real-time position of a vehicle is crucial. However, GPS signals often face "blind spots" or delays. This project solves that problem by using **Numerical Analysis** to estimate the train's position based on departure time and cumulative distance.

This algorithm compares two methods:
1.  **Linear Interpolation**: Assumes the train moves at a constant speed (Naive approach).
2.  **Lagrange Interpolation (Polynomial)**: Accounts for acceleration and deceleration phases, resulting in a more physically accurate curve.
